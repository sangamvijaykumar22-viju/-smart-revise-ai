Key Concepts Summary

1. Mutable vs Immutable

Lists are mutable, meaning you can change, add, and remove elements after creation. Tuples are immutable.

2. List Slicing

Access ranges of elements using list[start:stop:step].

colors = ['red', 'blue', 'green', 'yellow']
print(colors[1:3]) # Output: ['blue', 'green']

Common Pitfalls
1. Mutable Default Arguments

Avoid using [] or {} as default function arguments; they persist across function calls.

2. Modifying while Iterating

Removing items from a list while looping over it can cause unexpected behavior or skipped elements.

3. `is` vs `==`

Use == for value equality and is for identity (same memory location).

4. List Copying

list_b = list_a only creates a reference. Use list_a.copy() for a shallow copy.
